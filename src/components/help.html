功能介绍

Yunser Draw 是一款基于浏览器的图像编辑工具，可用于简单的涂鸦和图片编辑。

本软件具有如下功能：

  • 支持铅笔和蜡笔两种画笔。
  • 支持模糊、浮雕、灰度等 7 种滤镜。
  • 支持直线、圆和矩形的绘制。
  • 支持钢笔工具。
  • 支持两个图层
  • 支持过程回放（钢笔工具暂不支持回放）
  • 其他等工具。

使用帮助：

选择画笔或橡皮檫工具时，在画布上右键可以进行相关设置。

使用橡皮檫工具时，可通过鼠标滚轮来设置橡皮檫的大小

钢笔工具用来绘制曲线

使用铅笔工具或橡皮檫工具时，可以通过鼠标滚轮调整大小，右键可以进行设置。

快捷键

Ctrl + S：保存

Ctrl + Y：重做

Ctrl + Z：撤销

浏览器要求

尽可能使用最新的浏览器
